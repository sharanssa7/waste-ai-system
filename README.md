# ♻️ Waste AI Classification System

An AI-powered waste classification system that detects and classifies waste as **Biodegradable** or **Non-Biodegradable** using Computer Vision and Deep Learning.

---

## 📌 Project Overview

This system uses:

- YOLOv8 for object detection
- TensorFlow/Keras model for classification
- Flask for backend API
- HTML frontend for user interaction

The application allows users to upload waste images and automatically classify them using trained AI models.

## 🚀 How It Works

1. User uploads an image
2. YOLO detects waste object
3. Image is preprocessed
4. Keras model classifies waste
5. Result is displayed on UI
